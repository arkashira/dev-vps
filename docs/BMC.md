# Business Model Canvas – dev‑vps

| **Section** | **Details** |
|-------------|-------------|
| **Value Proposition** | • **Developer‑first managed VPS** – pre‑configured, secure, and auto‑scalable Linux instances.<br>• **Zero‑ops** – automatic patching, backups, monitoring, and instant rollback.<br>• **Rapid onboarding** – one‑click deployment from the Axentx dashboard, with pre‑installed dev stacks (Node, Python, Go, Rust, Docker, Kubernetes, etc.).<br>• **Integrated CI/CD** – built‑in GitHub Actions runner, auto‑deploy on push, and instant preview URLs.<br>• **Cost predictability** – pay‑as‑you‑go with monthly caps and free tier for hobby projects.<br>• **Security & compliance** – hardened kernels, SELinux/AppArmor, DDoS protection, and GDPR‑ready data residency options. |
| **Customer Segments** | • **Solo developers & freelancers** needing a quick, low‑maintenance environment.<br>• **Startup engineering teams** that want to avoid on‑prem infra overhead.<br>• **Enterprise devops teams** looking for a sandbox for prototyping or CI/CD pipelines.<br>• **Educational institutions** and coding bootcamps requiring isolated, disposable workspaces for students.<br>• **Open‑source contributors** who need reproducible environments for PR testing. |
| **Channels** | • **Axentx Platform** – integrated within the existing Axentx dashboard and API.<br>• **Marketplace listings** – GitHub Marketplace, Docker Hub, and Cloud Marketplace (AWS, GCP, Azure).<br>• **Community outreach** – blog posts, webinars, and sponsorships at dev conferences.<br>• **Referral & affiliate programs** – incentivize current users to bring new teams. |
| **Revenue Streams** | • **Subscription tiers** – Free (limited resources), Pro ($20/mo per instance), Team ($50/mo per instance + 10% discount for >5 instances).<br>• **Pay‑as‑you‑go** – hourly usage for ad‑hoc or burst workloads.<br>• **Add‑ons** – premium support, dedicated IP, extra backup retention, and custom domain mapping.<br>• **Enterprise contracts** – on‑prem or hybrid deployments with SLA guarantees. |
| **Cost Structure** | • **Infrastructure** – cloud compute, storage, networking, and CDN costs (AWS EC2, S3, CloudFront, etc.).<br>• **Operational tooling** – monitoring, logging, patching, backup services.<br>• **Development & maintenance** – core dev‑vps codebase, integration with Axentx BRAIN, and continuous improvement.<br>• **Customer support** – ticketing system, live chat, and knowledge base.<br>• **Marketing & sales** – content creation, community events, and partner programs. |
| **Key Resources** | • **Dev‑vps codebase** (open‑source repo `arkashira/dev-vps`).<br>• **Axentx BRAIN (pgvector)** – shared knowledge base and self‑learning loops.<br>• **Cloud provider accounts** (multi‑cloud strategy).<br>• **Automated CI/CD pipelines** for rapid feature rollout.<br>• **Security & compliance certifications** (ISO 27001, SOC 2). |
| **Key Activities** | • **Infrastructure provisioning & scaling** – automated VM lifecycle management.<br>• **Security hardening & patch management** – automated updates and vulnerability scanning.<br>• **Feature development** – new dev stacks, integration with CI/CD tools, and UI/UX improvements.<br>• **Customer onboarding & support** – documentation, tutorials, and responsive helpdesk.<br>• **Data collection & analytics** – usage metrics to inform pricing and roadmap. |
| **Key Partners** | • **Cloud providers** – AWS, GCP, Azure for compute & storage.<br>• **CI/CD platforms** – GitHub Actions, GitLab CI, CircleCI for integration.<br>• **Security vendors** – Qualys, Snyk for vulnerability scanning.<br>• **Payment processors** – Stripe, PayPal for subscription billing.<br>• **Developer communities** – FreeCodeCamp, Dev.to, and Stack Overflow for outreach. |
